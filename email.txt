### 🛠️ Tech Stack & Architectural Choice
While the assignment recommended React/Next.js, I chose to implement this suite using **Native PHP 8.0 and Vanilla JavaScript (ES6+)**. 

**Why Native?**
1.  **Zero Dependency Bloat:** No overhead from heavy frameworks or thousands of sub-packages.
2.  **Unmatched Performance:** Instantaneous load times and near-zero execution latency.
3.  **Future-Proofing:** Built on core web standards that remain stable for decades.
4.  **Full Control:** Surgical precision in UI implementation and state management.

### ✨ Key Features Delivered
